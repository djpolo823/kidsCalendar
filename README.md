# kidsCalendar
KidsCalendar es una app educativa que ayuda a organizar rutinas infantiles mediante un sistema de actividades y recompensas con estrellas. Fomenta hábitos saludables, autonomía y motivación intrínseca a través del refuerzo positivo, permitiendo visualizar el progreso diario de forma clara, cálida y amigable para niños y familias.
